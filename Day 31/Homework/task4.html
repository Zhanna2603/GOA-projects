What is HTML?

HTML (HyperText Markup Language) is a standard markup language to design the structure of a webpage. HTML is the combination of HyperText and Markup Language. HyperText defines the internal links between webpages, and Markup language defines the layout and presentation of text and media.

Basic HTML Structure

The basic structure of an HTML document contains the few mandatory tags, that must be used to create a webpage. This structure must be followed to create any type of HTML document. Here is the basic structure of an HTML document −
Elements of HTML Basic Structure. The following are the basic tags that define the basic HTML structure −

  1. !DOCTYPE html This element defines the document type as HTML. This element must be written before writing any HTML document.
  2. html.../html The html tag is the parent tag for all HTML elements. Everything related to create an HTML document must be written inside the html tag. CSS, JavaScript, and jQuery must also be written inside this tag.
  3. head.../head The head tag is a container tag for all those elements that are not directly displayed on the webpage but required for the page functionalities. It contains meta tags (which are used for SEO purposes), title tag, script tags, etc.
  4. title.../title The title tag is used to define the title of the webpage that you can see in the browser's tab, bookmarks list, and search engine results. This tag is also very important for SEO purposes to help search engine to understand the content of the webpage.
  5. body...body The body tag is the container tag for all those elements, which represents the main content of a webpage that displays on the browser.
  6. h1.../h1 The h1 tag is one of the heading tags. It is the most important heading tag, which defines the main title or headline of the webpage. Any text written inside h1 and /h1 is a top-level heading of the content.
  7. p.../p The p tag defines a paragraph, anything written inside p and /p displays as a paragraph on the webpage. Use multiple p tags to display text in different paragraphs.

HTML Video

Introduction to embedding videos in HTML Embedding videos in HTML is an important skill for any web developer. Video can significantly enhance the user experience on your site by providing visual content that can be more informative and engaging than text or images. In this article, we will look at the basic methods of inserting videos into HTML, including using the video tag, adding attributes to control videos, as well as inserting videos from popular platforms such as YouTube and Vimeo.
Videos on web pages can perform many functions, from educational content to entertainment videos and promotional materials. It is important to understand how to insert a video correctly and effectively so that it is displayed correctly on all devices and in different browsers. In this article, we will take a detailed look at all aspects of embedding videos in HTML so that you can confidently use this tool in your projects.
The video tag also supports multiple video sources via nested source tags. This allows you to specify multiple video formats to ensure cross-browser compatibility. For example, you can add videos in WebM format for browsers that do not support MP4.

Adding Attributes to Control Video

The video tag supports a variety of attributes that allow you to customize the behavior and appearance of your video. Here are some of them:
autoplay: Automatically plays the video when the page loads.
loop: Loops the video, playing it again when it finishes.
muted: Mutes the video by default.
poster: Sets a splash image that appears before the video starts playing. These attributes allow you to flexibly customize your video depending on the needs of your project. For example, the autoplay attribute can be useful for commercials that should start immediately after the page loads, while the loop attribute is useful for video backgrounds that should play continuously.
